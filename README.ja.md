# egadv - ノベルゲームづくりからのJavaScript入門

egadvは、ノベルゲームの作成を通してJavaScriptの基礎を学べるフレームワークです。簡単なウェブアプリケーションを作りながら、プログラミングの基礎を理解できます。

## デモ
リポジトリ内の各ステップのデモが利用できます。

## 機能
- 背景の表示と切り替え
- テキストの表示と選択肢の表示
- 立ち絵の表示
- 地図の表示
- ナビへのリンク
- ランダム関数の利用

## 必要環境
特にありません。最新のウェブブラウザがあれば動作します。

## 使い方
リポジトリ内のステップごとのHTMLファイルを開くと、デモを見ることができます。各ステップでは、JavaScript の基本的な構文を学習できます。

## データ・API
FIND/47の写真オープンデータや、Geo3x3ライブラリを利用しています。
